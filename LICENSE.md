BGPL - Better GNU General Public License.
BGPL is an open-source license that prioritizes credit over sharing.
The rules for BGPL are specific and cannot be changed.
The rules are:

1. Credit
 * Always required, no exceptions.
 * 1-3 lines minimum in the main configuration file or folder.
 * Must be visible and readable, NOT hidden.

2. Sharing
 * Always encouraged but not always required.
 * It is required if the code is modified past the line threshold set by the original creator (between 30 to 150 lines, the creator decides.)
 * It is also required if it is used commercially AND modified past the threshold.

3. Commercial Use
 * Credit is always required and is not optional.
 * If it was used in physical products, it's required for the maintainer to put a visible sticker or label crediting the creator.
 * If it was used in digital products, it's required for the maintainer to credit the creator in an about page of the product, in its description, or a linked webpage.
 * If it was used in advertisements, posters, or banners, it's required to credit the creator in any corner, and it must be visible and readable.

4. Fork Chain
 * A modification of a project can be declared as a fork of your own if it crossed the line threshold the creator sets.
 * The crediting of the creator must credit the last two forkers in the chain if any fork existed of the project.
 * The Original Creator sets the line threshold after using the license and it cannot be changed unless the Original Creator changes the threshold.
 * You can set the modified line threshold for declaration of a fork of the project at the end of the license text.

5. Private Use
 * A truly private use requires nothing but it must not be shared with the public, only with your team or friends but NOT with the public.
 * If it touches any commercial context, Rule 3 applies.

6. License Switching
 * You may switch to GPL or LGPL if needed.
 * However credit rules from Rule 1 and Rule 4 still apply regardless of the license switch.
 * Switching licenses does not exempt you from crediting the original creator.
